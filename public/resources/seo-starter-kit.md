# SEO Starter Kit

A quick-start guide to help you optimize your website for search engines.

## 1. Keyword Research
- Use tools like Google Keyword Planner or Ubersuggest
- Identify primary and secondary keywords

## 2. On-Page SEO
- Use keywords in titles, headings, and content
- Write unique meta descriptions
- Use descriptive alt text for images
- Create SEO-friendly URLs

## 3. Technical SEO
- Ensure your site is mobile-friendly
- Improve page speed
- Use HTTPS
- Submit a sitemap to Google Search Console

## 4. Content Strategy
- Publish high-quality, original content
- Update content regularly
- Use internal and external links

## 5. Tracking & Analytics
- Set up Google Analytics
- Monitor keyword rankings
- Track site performance

---

*Use this kit as a starting point for your SEO journey!*
